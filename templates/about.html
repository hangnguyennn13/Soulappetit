What We Do

Encourage by Facebook tag, we built an application that can guess the emotional states of people inside the image.

To accomplish that, we collected images from Google - with 3 main emotions: Happy - Emotionless - Unhappy. Then we train a model using these images, add that model to Flask app and then deploy it to google cloud for public share.
